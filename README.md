# Caesar Cipher Encryption Tool

A Python program to encrypt and decrypt text messages using the Caesar Cipher algorithm.

## 📌 Task
This project was completed as Task 1 of my Cyber Security Internship at **SkillCraft Technology**.

## 🔐 About Caesar Cipher
The Caesar Cipher is a classic encryption technique where each letter in the message is shifted by a fixed number of positions in the alphabet.

## ⚙️ Features
- Encrypt any text message
- Decrypt previously encrypted text
- User-defined shift value
- Handles uppercase and lowercase letters

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Clone this repository
2. Run the script:
3. Choose Encrypt or Decrypt
4. Enter your message and shift value

## 📖 Example
Input: HELLO, Shift: 3
Output: KHOOR
